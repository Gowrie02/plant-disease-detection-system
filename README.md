# Plant Disease Detection System

**Privately Funded Project** | ₹50,700 | In Development

An embedded AI system for real-time plant disease detection and classification using Raspberry Pi and CNN.

---

## Overview

This project aims to help farmers detect plant diseases at an early stage using computer vision and deep learning. The system captures images in the field and provides instant classification.

**Funding**: Secured ₹50,700 from United Soft Tech Pvt. Ltd.

**Role**: Lead Developer

---

## Features

- Real-time image capture and disease classification
- Embedded hardware integration (Raspberry Pi + Camera)
- Wireless data transmission using ESP8266
- Sensor integration (Arduino) for environmental monitoring
- CNN-based deep learning model for classification

---

## System Architecture

- **Hardware**: Raspberry Pi + Camera Module + Arduino Sensor Nodes + ESP8266
- **Software**: Python, TensorFlow/Keras, CNN Model
- **Workflow**: Image capture → Preprocessing → CNN Classification → Alert/Result
